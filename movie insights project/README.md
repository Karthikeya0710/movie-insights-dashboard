# Movie Insights Dashboard

This project analyzes an IMDB movie dataset and presents insights using an interactive dashboard.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Streamlit

## Features
- Movies released per year analysis
- Top movie genres
- Language distribution
- Top rated movies
- Popularity vs rating analysis
- Most voted movies
- Genre trends over time
- Interactive Streamlit dashboard
- Year filtering

## Dataset
IMDB Top Rated Movies Dataset from Kaggle

## Run the Dashboard

pip install streamlit pandas matplotlib

python -m streamlit run dashboard.py
